GST அமைப்பின் கீழ் பதிவு செய்ய வழிகள்

குறிப்பு 1:

20 லட்சத்துக்கு மேல் நிகர வருட வருமானம் வரும் மேல் சொன்ன அணைத்து வரி தாரர்களும் GST வரியில் உள்ளடங்குவர்.

குறிப்பு 2:

அ) GST இணைய தளத்தில் பதிவு செய்யும் முன் வட்டார அரசு அதிகாரியை அணுகி முதல் முறை தேவையான Provisional User ID மற்றும் Password பெற்றுக் கொள்ள வேண்டும்.

ஆ) Provisional User ID மற்றும் Password பெற்றுக் கொண்ட பிறகு http://www.gst.gov.in என்ற இணைய தளத்திற்கு சென்று அதில் கொடுக்கப்பட்ட வழிகாட்டல் படி படிவங்களை நிரப்ப வேண்டும்.

குறிப்பு 3:

படிவங்களை நிரப்பும் முன் கீழே உள்ள ஆவணங்களை தயார் செய்து கொள்வது நல்லது.

- குறிப்பு 2ல் உள்ளவாறு Provisional ID மற்றும் Password.

- ஈமெயில் முகவரி

- மொபைல் எண்

- வங்கி கணக்கு எண் மற்றும் IFSC குறியீடு கொண்ட வங்கி கணக்கு புத்தகம் அல்லது வங்கி அறிக்கை.

- கூட்டு ஒப்பந்த நகல் அல்லது நிறுவன பதிவு சான்றிதழ்

- நிறுவன உரிமையாளர்களின் ஒவ்வொருவரது புகைப்படம்

-GST தொடர்பான நடவடிக்கைளை மேற்கொள்ளும் அலுவலருக்கு அனுமதி கொடுக்கப்பட்ட சான்றிதழ்

- மேல் சொன்ன அலுவலரது புகைப்படம்

குறிப்பு 4:

இணைய தளத்தில் படிவங்களை நிரப்பிய பிறகு, ஆவணங்களை இணைய தளத்தில் ஏற்ற வேண்டும். அதன் பிறகு ஒப்புகை எண் ஒன்று விண்ணப்பதாரரின் மின் அஞ்சலுக்கு வந்து சேரும். இதனை GSTயின் மற்ற நடவடிக்கைகளுக்கு பயன்படுத்திக் கொள்ள வேண்டும்.
